# 📊 Zudio Sales Dashboard using Microsoft Excel

An interactive sales dashboard built using Microsoft Excel to analyze Zudio sales performance across different states, product categories, and store types.

---

## 📌 Project Overview

This project analyzes Zudio sales data using Pivot Tables, Charts, and Slicers to generate interactive business insights.

---

## 🎯 Objectives

- Analyze total sales
- Compare state-wise sales performance
- Analyze clothing categories
- Compare store types
- Build an interactive Excel dashboard

---

## 🛠️ Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Visualization

---

## 📊 Dashboard Features

- Total Sales Analysis
- Quantity Analysis
- Clothing Type Analysis
- Store Type Analysis
- Category Analysis
- Interactive Filters

---

## 📁 Dataset

The project uses a retail sales dataset containing approximately **5,000 sales records** with product, pricing, quantity, store, and regional information.

## 💡 Skills Demonstrated

- Data Cleaning

- Pivot Charts
- Dashboard Design
- Data Visualization
- Business Analysis
